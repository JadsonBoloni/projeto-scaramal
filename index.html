Demonstração Prática de Comandos SQL

Use os controles abaixo para simular INSERT, ALTER, DROP/DELETE e SELECT em uma tabela de usuários.

Tabela de Usuários (Simulação de Banco de Dados)

ID Nome Idade

Comandos SQL

Inserir um novo usuário

Exemplo: INSERT INTO usuarios (nome, idade) VALUES ('gabriel', 25);

Adicionar uma coluna à tabela

Exemplo: ALTER TABLE usuarios ADD cidade VARCHAR(50);

Remover um usuário por ID

Exemplo: DELETE FROM usuarios WHERE id = 1;

Deletar a tabela inteira

Exemplo: DROP TABLE usuarios;

SELECT: Filtrar e exibir usuários

Exemplo: SELECT * FROM usuarios WHERE idade > 20;
